#  Getting Started

### Clone Repository

```bash
git clone https://github.com/<your-username>/<repository-name>.git
```

### Install Dependencies

```bash
pip install -r src/requirements.txt
```

### Start MLflow UI

```bash
mlflow ui
```

Open:

```text
http://localhost:5000
```
